Zima

Zima

Reimagined Zima, a classic lemon-lime beverage brand, with branding, packaging, 3D modeling, web design, and advertising. The project modernizes the brand while keeping its unique personality.

Project Period

10 Weeks

Role

Graphic Designer / Package Designer / Motion Designer / Web Developer

Tools

VS Code / Photoshop / Illustorator / Cinema 4D / After Effects

Collaborate With

Hannah Silva

*This work is a student project created for educational purposes only. All trademarks, logos, and brand assets are the property of their respective owners.

Challenges

A major challenge was newly learning and applying the modeling and material-creation workflow in Cinema 4D. Since this was my first project using this software in a practical context, I needed to understand the interface, experiment with different material properties, and refine the modeling process through continuous trial and error.

In addition, I aimed to incorporate visual elements inspired by the name “Zima,” which means “winter” in several languages, and develop a snowflake motif that would feel modern rather than cliché. Balancing this concept with a clean and cohesive brand identity required constant iteration, design refinement, and thoughtful decision-making throughout the project.
