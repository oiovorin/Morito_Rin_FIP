Loop Earbuds Pro Case Study

Loop Earbuds Pro

Developed a concept wireless earbud for Loop, an existing brand. The project included 3D modeling, 3D animation, and web development using model-viewer, GSAP, and an X-ray slider, showcasing a sleek, modern, and user-friendly design.

VISIT WEBSITE →

Project Period

8 Weeks

Role

Graphic Designer / Motion Designer / Web Developer

Tools

VS Code / Photoshop / Illustorator / Cinema 4D / After Effects

*This work is a student project created for educational purposes only. All trademarks, logos, and brand assets are the property of their respective owners.

Challenges

This project began with researching the mechanisms of typical wireless earbuds and progressed to designing both the internal system and the external appearance, resulting in a realistic earbud design that could function if it actually existed.The biggest challenge was creating a 3D model that matched the image I had envisioned. Due to the complexity of the shape, modeling was difficult and required numerous rounds of trial and error.

On the website, I incorporated a model viewer and an X-ray slider to create an interactive experience for users.
